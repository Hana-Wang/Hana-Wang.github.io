Publications

  1. Counting via Markov-type technique. (Work paper) (2020).
  • In this study, we have developed an unusual stochastic model using markov-type technique
    and use this model to solve the specified counting problem. In this model, we consider
    counting restricted paths in a discrete time model, motivated by markov chain knowledge, we
    develop a new transform probability matrix, the matrix with different dimensions according
    to different restrictions can solve diferent counting problem. Considering the computer
    efficiency, we use sparse matrix to represent the transform matrix in MATLAB and solve the counting problem.
  1. Graphical Theoretical Construction of Discrete time Share Price Paths from Matroid. ICACSM 2020: XIV. International Conference on Analytic Combinatorics and Symbolic Methods Vol:14, No:4, 2020.
  • In this study, we have developed an algorithm to construct share price paths from general
    binary trees. We also have provided a new relationship between lattice integer points and
    tutte polynomials of transversal matroid. Use this new way of connection together with the
    algorithm, we are able to construct a share price path from a given connected lattice path
    matroid.
  1. Path Calculations and Option Pricing. Doctoral dissertation, DOI: 10.25392/LEICESTER.DATA.11807622.V1 (2020).
